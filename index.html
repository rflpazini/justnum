.justnum();

Um plugin para bloquear usuários de digitar números

Star Fork

Instalação

Instale via Bower

$ bower install justnum

ou

Baixe o .zip

.justnum();

Sabe quando você quer que o usuário digite apenas números em um campo? Este plugin jQuey faz isto pra você.

No demo asseguir você irá notar que assim que inserido, uma letra ou caracter, que não seja um número será exibido um alert mostrando para o usuário que não é possível esta inserção no campo escolhido. 

                        

<input id="example-just" type="text" class="form-control" placeholder="Digite alguma coisa ">

<script>
    $('#example-just').justnum();
</script>

Mas cade os pontos e as vírgulas?

Caso você precise que o usuário interaja colocando pontos ou vírgulas, basta ativar a opção allow disponível no plugin.

                        

<input id="example-dot" type="text" class="form-control" placeholder="Use . e , (1,32.9)">

<script>
    $('#example-dot').justnum({
        allow: true
    });
</script>

Precisa destruir o evento?

Sem problemas, caso seja necessário é só passar o parametro 'destroy'.

                        

<input id="example-destroy" type="text" class="form-control" placeholder="Aqui o lero-lero funciona">

<script>
    $('#example-destroy').justnum('destroy');
</script>

Propriedades e parâmetros

Propriedade Tipo O que é
allow boolean Deixa que o usuário insira pontos "." e vírgulas "," nos campos
'destroy' destroy Desabilita a função do componente

Uso

1. Importe o jQuery para seu projeto

                
<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>

2. Importe o justnum

                
<script src="bower_components/justnum/dist/justnum.min.js"></script>

3. Escolha o elemento em que quer aplicar o plugin

                
<script>
    $('input').justnum();
</script>
